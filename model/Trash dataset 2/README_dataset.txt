Trash Sorting Assistant augmented balanced dataset

Included classes: cardboard, glass, metal, paper, plastic
Excluded class: trash, because it contains mixed/overlapping waste items.

Target images per class: 600
Augmentation used: lighting changes, contrast/color changes, rotations, flips, blur, shadows, and dirty/smudge effects.

- glass: 501
- metal: 410
- paper: 594
- plastic: 482
